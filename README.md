# Proyecto CLV (Customer Lifetime Value)

El proyecto CLV tiene como objetivo medir y analizar el valor económico de los clientes a lo largo del tiempo mediante la integración de datos de facturación y uso de productos. A partir de fuentes como `facturacion mensual`(ingresos) y `productos mensuales` (servicios/engagement), se construye un modelo que permite entender cuánto aporta cada cliente al negocio, cómo evoluciona su valor y qué tan profunda es su relación con los productos. Este enfoque permite soportar decisiones estratégicas enfocadas en retención, crecimiento y segmentación.

El pipeline integra, limpia y consolida la información a nivel de cliente, periodo y línea de producto, asegurando consistencia y evitando duplicaciones en métricas clave como ingresos y RGU. Como resultado, se genera un dataset analítico con indicadores como facturación, cantidad de productos, antigüedad y convergencia, que permite identificar clientes de alto valor, detectar riesgos de churn y encontrar oportunidades de upselling o cross-selling, además de servir como base para modelos predictivos y análisis avanzados.
